Description

Blocchi is a logic/puzzle game, in which you have to fill up the game field with different shapes built up from maximum 9 blocks using different transformations.
In two different game modes you can compete to last the longest possible (where the time to fill up the game filed is getting shorter and shorter each level) or to reach the lowest score in the shortest time.

System Requirements

  • Microsoft Windows 2000 (or later)
  • DirectX 8 compatible videocard and soundcard
  • PS/2 or USB compatible mouse and keyboard
